##Cadastro Usuário

### POST /usuario
````json
{
  "email" :"lurregia@zup.com",
  "nomeUsuario" : "Lurregia Barreto",
  "timeCoracao" : "ATLETICO_MG"
}

````

- Status 201
````json

  "nomeUsuario": "Lurregia Barreto",
  "timeCoracao": "CORINTHIANS",
  "pontos": 0
}
````
